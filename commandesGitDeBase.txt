commande Git de base

1. Pour commencer, créer un dépot en ligne sur github si ce n'est pas deja fait.
2. Copier l'adresse du dépo via le bouton "clone or Download"
3. Taper dans le terminal : git clone "adresse"
4. Faire nos modifs
5. Demander à git de suivre nos nouveau fichiers avec : git add*
6. Demander à git de faire une sauvegarde de nos fichiers
7. Demander à git d'envoyer les mise a jour en ligne sur github
